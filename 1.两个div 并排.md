### #块级元素水平居中

设置 margin-left:auto 以及 margin-right:auto;

或设置 margin: 0 auto;

 

button 的默认样式中有 **box-sizing:border-box**

即 button 元素的 width / height = padding + border + 内容的宽度/高度。

内容的 宽度 / 高度 随 padding / border 的改变而改变。
